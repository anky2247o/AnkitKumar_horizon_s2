# Task 1: Rover Travel Time Calculator

This Python script calculates the time required for a rover to travel between two points.

## How to Run
Ensure you have Python 3 installed, then run:
```bash
python3 rover_calc.py
```

## Physics Logic
The script calculates the Euclidean distance between origin and destination. It then determines if the rover reaches the destination while still accelerating or if it hits the **Top Speed** first and cruises the remaining distance.

## Example Output
* **Distance**: 141.4 m
* **Time**: 19.1 seconds
